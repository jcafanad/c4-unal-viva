# pres_C4_AFANADOR

Viva submission — C4 UNAL, Tema 7 (Crowdsourcing).

Frames the Iguaque-Merchán páramo delimitation as a multi-agent
crowdsourcing problem, through UCB1, DIG, and DEC to expose what each
algorithm's own axioms silently assume about the actors' "incidencia
territorial". Closes by revealing that "incidencia territorial" came
from a Dung/σ* argumentation framework over the same actors all along
— the bandits never knew.

## Files

- `pres_C4_AFANADOR.html` — static rendered view, open directly in a
  browser, no setup required.
- `pres_C4_AFANADOR.py` — the marimo notebook source.
- `layouts/pres_C4_AFANADOR.slides.json` — slide-layout configuration
  referenced by the notebook.
